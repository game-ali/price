User Guide for Tag Price Generator

Introduction

Welcome to the Tag Price Generator, your go-to tool for creating and managing product price tags. This guide will help you navigate the application and make the most out of its features. Whether you're adding items, editing descriptions, or managing images, this guide has you covered.

Getting Started

To begin, open the application in your browser. You'll see the following sections:

  • Company Information: Fill in details like company name, description, contact info, and location.
  • Item Information: Add product details, including name, image, price, and structured descriptions.
  • Interactive Table: View added items in a table format, where you can edit or delete entries dynamically.

Adding Items

Step-by-Step Guide

  1. Fill in the fields:
    • Item Name: Enter the name of your product.
    • Upload Image: Select an image file from your device (e.g., .jpg, .png).
    • Item Description: Write a structured description in the format key=value, separating each property with a new line. For example: 
      memory=8GB
ram=7GB
opt=Good
    • Item Price: Specify the price of the item.
  2. Submit the Item: Click the Add item button. Your item will appear in the table below.

Editing Items

What You Can Edit

  • Text Fields: Item names, descriptions, and prices.
  • Images: Replace product images with new ones.
  • Description Items: Modify individual key-value pairs within the description.

How to Edit

  1. Click the Edit button at the top of the page.
  2. Follow the prompt: "Click on the cell, image, or description item you want to edit."
  3. Click the target item:
    • Text Fields: An input box will appear for editing.
    • Images: A file upload dialog will let you select a new image.
    • Description Items: Click the specific key-value pair (e.g., memory=8GB) to edit it.
  4. Press Enter or click outside the input box to save changes.

Deleting Items

What You Can Delete

  • Entire rows in the table.
  • Individual description items (key-value pairs).
  • Images.

How to Delete

  1. Click the Delete button at the top of the page.
  2. Follow the prompt: "Click on the cell, image, or description item you want to delete."
  3. Click the target item:
    • Text Fields: The cell content will be cleared.
    • Images: The image will be removed.
    • Description Items: The key-value pair will be deleted.
  4. Confirm the deletion if prompted.

Special Features

  • Structured Descriptions: Use the key=value format for cleaner, more readable descriptions.
  • Dynamic Image Handling: Replace outdated images or delete them entirely.
  • Interactive Table: The table updates dynamically as you add, edit, or delete items.

Tips for Best Use

  • Use Clear Descriptions: Stick to the key=value format for consistency.
  • Optimize Images: Use images with dimensions around 100x100px for best results.
  • Regular Updates: Keep your product data current by editing or deleting outdated information.
  • Backup Your Data: Copy your data to a spreadsheet or save it elsewhere for safekeeping.

Troubleshooting

Common Issues

  • Problem: The "Add item" button doesn’t work.
    Fix: Ensure all fields are filled out, including uploading an image.
  • Problem: Description items don’t format correctly.
    Fix: Use the key=value format and ensure each pair is on a new line.
  • Problem: Images don’t display or can’t be replaced.
    Fix: Ensure your image file is in a supported format (e.g., .jpg, .png).
  • Problem: Deleting or editing doesn’t work.
    Fix: Ensure you’ve clicked the Edit or Delete button first.

Conclusion

The Tag Price Generator is a versatile tool for managing product information. Follow this guide to add, edit, and delete items effectively. Keep your data organized and up-to-date for the best results. Happy tagging!
